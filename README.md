# 90 Anos da Maria Luisa - Foto Restaurada

Um aplicativo simples, elegante e nostálgico criado para exibir e baixar a foto restaurada da família em celebração aos 90 anos da Maria Luisa.

## Como adicionar a sua foto real

1. Pegue a foto restaurada que você tem no seu computador.
2. Renomeie o arquivo da foto exatamente para `familia-restaurada.png`.
3. Coloque este arquivo dentro da pasta `public/` na raiz deste projeto (substituindo ou colocando ao lado do arquivo de texto que está lá).
4. O aplicativo automaticamente carregará a sua imagem!

## Como rodar localmente

Se você baixou o código para o seu computador, certifique-se de ter o Node.js instalado e rode:

```bash
npm install
npm run dev
```

## Publicando no GitHub e Vercel/Netlify

Como este projeto usa Vite e React, você pode facilmente publicá-lo:
1. Faça o commit de todos os arquivos (incluindo a sua foto na pasta `public`) e faça o push para o seu repositório no GitHub.
2. Conecte o repositório na Vercel, Netlify ou GitHub Pages.
3. As configurações de build padrão (`npm run build` e pasta `dist`) funcionarão automaticamente.
